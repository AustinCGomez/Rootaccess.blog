Welcome to `rootaccess.blog`.

This is where I write about IT infrastructure, security, Linux, homelab experiments, and whatever else comes up at the command line.

## How this works

Posts live in `_posts/` as Markdown files. The filename format is:

```
YYYY-MM-DD-your-post-title.md
```

Each post starts with front matter like this:

```yaml
---
layout: post
title: "Your Post Title"
date: 2024-06-01 00:00:00 -0600
tags: [linux, security]
excerpt: "One sentence describing what this post covers."
---
```

Then write your content in Markdown below the `---`.

## Useful tags

Keep tags consistent so filtering works cleanly. Suggested tags:

- `infrastructure`
- `security`
- `linux`
- `networking`
- `how-to`
- `homelab`
- `misc`

Delete this post when you're ready to go live.
